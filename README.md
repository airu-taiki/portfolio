# 切り抜き動画作成ソフト

## これは何？
YouTubeの動画を切り抜きした上で、連結させて１つの動画にするプログラム

プログラム流れ
- 秒数指定
- 動画出力
- 出力から切り抜き
- 繰り返し
- 各動画連結
- 連結したものを出力
大体こんな感じ

## やること
- できるかどうか検証
- できそうなら制作開始
- 時期を決める
    - 早めにプロトタイプを作成
    - 余裕があればGUIも作成
    - 冒頭、動画の間、エンディングに任意のファイルを入れるオプションなども可能であれば実装
